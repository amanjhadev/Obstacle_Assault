# 🎮 Obstacle Assault

A 3D obstacle course game developed in **Unreal Engine 5** using **C++**.

The objective of the game is to navigate through a series of moving and rotating obstacles while avoiding collisions and reaching the finish line. This project was built to strengthen my understanding of Unreal Engine's gameplay framework and C++ programming by implementing core gameplay mechanics from scratch.

---

## 📖 Overview

Obstacle Assault is a beginner-friendly game development project that focuses on the fundamentals of gameplay programming in Unreal Engine. Instead of relying solely on Blueprints, the gameplay systems are implemented in C++, providing a deeper understanding of how Unreal Engine manages actors, components, transforms, collision detection, and real-time updates.

Throughout this project, emphasis was placed on writing clean, modular, and maintainable code while following good object-oriented programming practices.

---

## ✨ Features

* Character navigation through a 3D obstacle course
* Moving platforms
* Rotating obstacles
* Collision detection
* Frame-rate independent movement using Delta Time
* Gameplay logic implemented in C++
* Modular actor-based architecture
* Clean and organized project structure
